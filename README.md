# Grade Clicker

В ходе данной лабораторной работы мы создали свое приложение на основе Jetpac Compose и познакомились с жизненым цикром Activity


## Цель работы

Целью лабораторной работы является изучение жизненного цикла Activity в Android. Также потрогать методы
onCreate(), onStart(), onResume(), onPause(), onStop(), onRestart(), onDestroy(). Ознакомление с логированием

## Контрольные вопросы

- onCreate() onStart() onResume()
- onPause() нужна когда активити чистично теряется чем-то извне, а он стоп когда активити полностью скрыта
- Когда активити вызывается после остановки
- Активити уничтожается и создается заново
- При rememberSaveable состояние сохраняется даже при повороте экрана
- Без этого система не может правильно настроить жизненный цикл и сбоит
- Чтобы логировать состояния, ошибки, варнинги и другие состояния ссистемы
